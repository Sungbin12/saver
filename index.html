제 3회 정기 점검 안내

2023.09.25
업데이트

안녕하세요, 안남초등학교 방송부 통합 홈페이지를 담당자 5학년 기술1입니다.

​

9월 27일 오전 8시부터 9월 28일 오전 8시까지 24시간 동안 정기 점검이 실시됩니다.
업데이트로 바뀌는 것은 다음과 같습니다.


■ 메인 페이지가 변경됩니다!
우리가 접속하면 제일 먼저 보는 메인 페이지가 업데이트 됩니다!
위에 메뉴에는 왼쪽 기준으로 로고, 로그인, 제보/문의, 민방위가 추가됩니다.
그리고 그 아래에는 공지사항(최대 2개) 그 밑에는 오늘의 일정이 새롭게 생깁니다!
오늘의 일정이 만약 등록되지 않았다면 '등록된 일정이 없습니다.' 라고 뜹니다!


■ 로그인 페이지 변경
메인 페이지가 변경됨에 따라 메인 페이지에 같이 있었던 로그인 페이지는 따로 분리되어, 다른 페이지에서 로그인을 진행하게 됩니다.
이러면 보안도 더 강해질 수 있겠죠?


■ 메인 페이지 하단 내용 변경
메인 페이지 하단에 '개인정보 처리 방침', '저작권', '안남초등학교 통합 홈페이지 저작권 표시'가 메인 페이지 하단에 새로 생깁니다.


■ 저작권, 라이선스 페이지 추가
위에 있는 '메인 페이지 하단 내용 변경'에 의해서 안남초등학교 방송부 통합 홈페이지 저작권과 산돌구름 폰트 라이선스 배너를 표시 할 수 있는 페이지가 추가됩니다.


■ 민방위 페이지 수정
민방위 페이지 상단 중앙에 빨간 글씨로 '

위 민방위 정보는 8월에 시행한 민방위 훈련 정보입니다.

' 이렇게 표시됩니다.
이렇게 표시 하는 이유는 이용자의 혼동을 막기 위해서 입니다.


■ 팝업 UI 변경
이번 10월 업데이트로 메인 화면이 바뀌며 팝업 UI도 변경되었습니다!
이전까지는 알 수 없는 이유로 오늘 하루 동안 보지 않기 기능이 작동 하지 않았지만, 이번 업데이트 UI 변경으로 작동하게 됩니다!


■ 코드, 보안 업데이트 진행
이번 업데이트로 코드 점검과 취약한 부분을 보안 업데이트 진행으로 쾌적한 홈페이지 환경을 만들기 위해 노력하겠습니다.


■ 업데이트 이후 안내
이번에는 9월 업데이트에 이어 9월 업데이트 때 못했던 기능과 업데이트를 진행하였습니다.
그래서 10월 업데이트는 다소 작은 규모로 진행됩니다.
그래도 메인 페이지와 로그인 페이지가 개선이 됨에 따라 버그와 페이지 이동의 오류는 없을 수 없습니다.
만약 그런 일이 일어난다면 제보/문의로 내용 보내주시면 검토 후, 해결 하도록 하겠습니다.


■ 이용 안내
업데이트 중에는 서비스 이용이 불가능하다며, 중요한 사항이 있어도 업데이트를 중간에 중단 할 수 없습니다.
업데이트 시간은 예정 시간이며, 예정보다 빨리 또는 느리게 진행 될 수 있습니다.
업데이트는 모든 파일 삭제 후, 파일을 작업용 PC에서 다시 등록하니 완전한 이용이 중지 됩니다. (오류 방지)


앞으로도 더 좋은 서비스를 제공하기 위해 지속적으로 개선해 나가겠습니다.

감사합니다.
