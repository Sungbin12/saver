제 1회 정기 점검 안내

2023.07.31
서비스

안녕하세요, 안남초등학교 방송부 통합 홈페이지를 담당자 박성빈입니다.

​

8월 1일 오전 8시부터 8월 2일 오전 8시까지 24시간 동안 정기 점검이 실시됩니다.
업데이트로 바뀌는 것은 다음과 같습니다.


■ 로그인 후, 화면 선택 부분 변경, 캘린더 수정
이때까진 로그인 전에만 공지사항을 확인 하실 수 있었지만, 이번 업데이트로 로그인 후, 화면 선택에서 공지사항을 확인 하실 수 있습니다.
그리고 이번 업데이트로 개인정보 처리방침이 새롭게 만들어졌습니다.
캘린더도 7월 일정에서 8월 일정으로 변경됩니다.

■ 코드, 보안 업데이트 진행
이번 업데이트로 코드 점검과 취약한 부분을 보안 업데이트 진행으로 쾌적한 홈페이지 환경을 만들기 위해 노력하겠습니다.

■ 이용 안내
업데이트 중에는 서비스 이용이 불가능하다며, 중요한 사항이 있어도 업데이트를 중간에 중단 할 수 없습니다.
업데이트 시간은 예정 시간이며, 예정보다 빨리 또는 느리게 진행 될 수 있습니다.
업데이트는 모든 파일 삭제 후, 파일을 작업용 PC에서 다시 등록하니 완전한 이용이 중지 됩니다. (오류 방지)


앞으로도 더 좋은 서비스를 제공하기 위해 지속적으로 개선해 나가겠습니다.

감사합니다.
