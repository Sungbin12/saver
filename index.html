제 2회 정기 점검 안내

2023.08.27
업데이트

안녕하세요, 안남초등학교 방송부 통합 홈페이지를 담당자 5학년 기술1입니다.

​

9월 1일 오전 8시부터 9월 2일 오전 8시까지 24시간 동안 정기 점검이 실시됩니다.
업데이트로 바뀌는 것은 다음과 같습니다.


■ 비상 연락망이 변경됩니다!
개인정보 유출이 우려되어 이름 중간 자리가 '*' 으로 변경됩니다. 그리고 전화번호 가운데 자리도 똑같이 적용되며, 뒷자리 2자리도 추가로 변경됩니다.


■ 모바일 버전 최적화
8월 업데이트로 모바일 접속이 완전히 금지되어 접속하기가 힘들어 사용에 불편함이 있었으나, 9월 개학으로 인해 평상시에 PC 이용이 어려워 사이트 이용이 매우 어려웠지만, 9월 업데이트로 변경됩니다. 하지만, 메타태그 추가로 최적화가 된 것으로 일부 기능에는 최적화가 제대로 되지 않을 수 있습니다.


■ 메인 페이지 (index) 로그인 보안 문제 강화
이제 메인 페이지 로그인이 홈페이지 내 html에 있는게 아닌 js 파일을 서버에서 불러와 읽는 형식으로 변경되었습니다.


■ 민방위 페이지 추가
민방위 훈련이 재개되면서 민방위 훈련 페이지가 추가 됩니다.


■ 방송부 캘린더 수정
8월 일정이 없어서 등록이 되지 않았지만 9월 일정이 새롭게 등록됩니다.


■ 방송부 캘린더 업데이트
이때까진 텍스트로만 볼 수 있었지만 이젠 이미지로 볼 수 있습니다!


■ 공지사항 일련번호 003 내용 삭제
○ 필요 없는 내용이 공지사항에 등록되어 있어 9월 업데이트로 삭제됩니다.
○ 삭제 시, 일련번호 003도 삭제가 같이 될 시, 이때까지 썼던 일련번호에 오류가 발생할 수 있으므로 게시물은 놔두돼, 내용을 삭제했습니다.


■ 공지사항 <상위> UI 변경
전체 공지 시, 이제 빨간 사각형 박스에 전체 공지라는 문구가 넣어집니다.


■ 제보 페이지 추가
이때까진 이메일에서만 제보가 가능 했지만 이번 9월 업데이트로 제보와 문의를 동시에 할 수 있는 홈페이지 문의 기능이 생겼습니다. 위 기능은 계속해서 업데이트가 될 예정이며, 서비스 종료 때까지 업데이트 예정입니다.


■ 404 페이지 추가
이때까진 페이지 링크를 잘못 입력하거나 없는 링크를 입력했을 때 깃허브의 기본 404 페이지가 노출 됐지만 이젠 안남초등학교 방송부 사이트만의 404 페이지가 추가됩니다!


■ 오류 페이지 개선
○ 10번이랑 비슷하지만 오류 페이지가 개선, UI가 변경됩니다.
○ 개선으로 바뀌는 오류 페이지는 다음과 같습니다. ---- 200,404,500,502,503,504
이 외의 페이지 : 사이트 자체 about:blank


■ 200 오류 코드 추가
업데이트 전 부분 업데이트 (1차 업데이트)시 index.html 삭제 시, README.md 페이지가 떴지만, README.md에 있는 메타태그 자동 이동 방식으로 200 오류 코드로 이동 후 내용 표출 방식으로 변경하였습니다.


■ 업데이트 시, 공룡 게임 추가
업데이트 시, 재미있게 업데이트 시간을 즐길 수 있도록 공룡 게임이 추가 되었습니다. - 추가 된 위 기능은 계속해서 업데이트가 될 예정이며, 서비스가 종료 될 때까지 업데이트 예정입니다. 하지만 매월 매월 업데이트 예정이 있진 않습니다.


■ 개인정보 처리방침 변경
개인정보 처리방침 변경 - 사이트 도메인 변경에 따라 도메인과 업데이트로 기능적인 부분이 변경됨에 따라 개인 정보 처리방침 변경


■ Chat GPT 도입
아래처럼 코드 보안을 점검 할 때, 신규 기능을 출시 할 때 기술적인 한계가 있을 때 사용할 수 있도록 변경되었습니다.


■ 코드, 보안 업데이트 진행
이번 업데이트로 코드 점검과 취약한 부분을 보안 업데이트 진행으로 쾌적한 홈페이지 환경을 만들기 위해 노력하겠습니다.


■ 업데이트 이후 안내
이때까지의 업데이트 중 제일 규모가 큰 업데이트입니다.
버그, 페이지 간의 이동 오류(속도 느림 X[페이지 간의 이동 오류는 아예 이동이 안되는 것]), 접속 불가 오류 등 해당 사항 발생 시, 제보/문의로 내용 보내주시면 검토 후, 해결 하도록 하겠습니다.


■ 이용 안내
업데이트 중에는 서비스 이용이 불가능하다며, 중요한 사항이 있어도 업데이트를 중간에 중단 할 수 없습니다.
업데이트 시간은 예정 시간이며, 예정보다 빨리 또는 느리게 진행 될 수 있습니다.
업데이트는 모든 파일 삭제 후, 파일을 작업용 PC에서 다시 등록하니 완전한 이용이 중지 됩니다. (오류 방지)


앞으로도 더 좋은 서비스를 제공하기 위해 지속적으로 개선해 나가겠습니다.

감사합니다.
